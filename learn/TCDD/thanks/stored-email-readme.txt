=== Store-Email ===

* This javascript will check cookie for 'stored-cookie' variable and will create it if not found with the value from the URL query variables 'email' or 'e'.


=== Installation ===

1. Upload the file stored-email.js to your server and take note of its URL.
2. Insert this code inside the <head> tag of the page you want to use Stored-Email.

	<script src="THE_URL_OF_THE_stored-email.js_File"></script>

3. Done.
